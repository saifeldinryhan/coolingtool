# Cooling Tools Pro | أدوات التبريد المحترفة

تطبيق ويب متقدم لحسابات وأدوات التبريد والتكييف، مع دعم البحث الذكي بالذكاء الاصطناعي عن معلومات الضواغط.

## ✨ الميزات الرئيسية

### 🔧 أدوات الحسابات
- **حساب قطر السلك**: حساب القطر المناسب للأسلاك الكهربائية
- **حساب التدفق من الإزاحة**: حساب معدل التدفق بناءً على إزاحة الضاغط
- **تحويلات الوحدات**: تحويلات شاملة لوحدات التبريد والضغط والحرارة والكهرباء

### 🌡️ حاسبات متخصصة
- **حساب الحرارة من المقاومة**: تحويل قراءات المقاومة إلى درجات حرارة
- **حسابات كهربائية**: حسابات الفولت والأمبير والواط
- **حسابات الغرف**: حسابات متعلقة بتكييف الأماكن المغلقة

### 🤖 البحث بالذكاء الاصطناعي
- **البحث عن معلومات الضواغط**: ابحث عن أي موديل ضاغط واحصل على معلومات شاملة
- **دعم نماذج ذكاء اصطناعي متعددة**: Gemini و OpenAI و Grok و DeepSeek
- **بحث محلي سريع**: نتائج فورية مع معالجة ذكية للبيانات

### 🎨 واجهة المستخدم
- **تصميم حديث وسهل الاستخدام**: واجهة ديناميكية مع تجربة مستخدم سلسة
- **دعم اللغة العربية**: واجهة كاملة باللغة العربية
- **الوضع الليلي**: دعم كامل للوضع الليلي والنهاري
- **تصميم متجاوب**: يعمل بشكل مثالي على جميع الأجهزة

## 📋 المتطلبات

- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- اتصال بالإنترنت (لاستخدام ميزات الذكاء الاصطناعي)

## 🚀 التثبيت والتشغيل

### الطريقة 1: النشر على GitHub Pages

1. انسخ المستودع إلى حسابك على GitHub
2. في إعدادات المستودع، اذهب إلى **Settings** → **Pages**
3. اختر **Deploy from a branch** وحدد الفرع `main` والمجلد `root`
4. سيتم نشر التطبيق تلقائياً على `https://yourusername.github.io/cooltools-github`

### الطريقة 2: التشغيل المحلي

```bash
# استخدام PHP
php -S localhost:8080

# أو استخدام Python 3
python3 -m http.server 8080

# أو استخدام Node.js
npx http-server
```

ثم افتح المتصفح وانتقل إلى `http://localhost:8080`

## ⚙️ الإعدادات المتقدمة

### تفعيل البحث بالذكاء الاصطناعي

لاستخدام ميزات البحث المتقدمة بالذكاء الاصطناعي، تحتاج إلى:

1. إنشاء ملف `.env` في جذر المشروع:

```env
GEMINI_API_KEY=your_gemini_api_key
OPENAI_API_KEY=your_openai_api_key
GROK_API_KEY=your_grok_api_key
DEEPSEEK_API_KEY=your_deepseek_api_key
```

2. تشغيل خادم PHP محلي:

```bash
php -S 0.0.0.0:8080
```

3. الوصول إلى التطبيق عبر `http://localhost:8080`

## 🔐 الأمان

- **حماية مفاتيح API**: لا تشارك مفاتيح API الخاصة بك
- **التحقق من العميل**: يستخدم البرنامج مفتاح عميل للتحقق من الطلبات
- **CORS محمي**: السماح فقط بالطلبات من النطاقات المصرح بها

## 🐛 استكشاف الأخطاء

### البحث بالذكاء الاصطناعي لا يعمل
- تأكد من وجود ملف `.env` بمفاتيح API صحيحة
- تحقق من سجل الأخطاء في `proxy_error.log`
- تأكد من أن خادم PHP يعمل بشكل صحيح

### الحسابات تعطي نتائج خاطئة
- تحقق من صحة المدخلات
- تأكد من اختيار الوحدات الصحيحة
- امسح ذاكرة التخزين المؤقت للمتصفح
